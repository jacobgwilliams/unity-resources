# Unity Project Settings Template
# Copy these settings to your Unity project's ProjectSettings folder

## Player Settings Recommendations

### Company Name
- Set a consistent company name for all builds

### Product Name
- Use descriptive names for different platforms
- Example: "MyGame" for PC, "MyGame Mobile" for mobile

### Version
- Use semantic versioning (e.g., 1.0.0)
- Increment appropriately for releases

### Icon
- Create platform-specific icons
- Recommended sizes:
  - PC: 256x256, 512x512
  - Mobile: 1024x1024 (iOS), 512x512 (Android)
  - Console: Platform-specific requirements

### Resolution and Presentation
- Set default screen width/height
- Enable/disable fullscreen mode as needed
- Set default screen orientation for mobile

### Splash Screen
- Configure splash screen settings
- Set logo and background
- Configure animation settings

### Other Settings
- Set bundle identifier (com.yourcompany.yourgame)
- Configure supported platforms
- Set scripting backend (IL2CPP recommended for mobile)
- Configure API compatibility level

## Quality Settings Recommendations

### Quality Levels
- Create custom quality levels for different platforms
- PC: High, Medium, Low
- Mobile: High, Medium, Low, Very Low
- Console: High, Medium

### Texture Quality
- Set appropriate texture quality for each level
- Consider memory constraints on mobile

### Anti Aliasing
- Enable MSAA for PC builds
- Use FXAA for mobile if needed

### Shadow Settings
- Configure shadow resolution and distance
- Use lower settings for mobile

### LOD Bias
- Adjust LOD bias for different quality levels
- Higher values for lower quality levels

## Physics Settings

### 2D Physics
- Set gravity (default: -9.81)
- Configure default material
- Set velocity iterations and position iterations

### 3D Physics
- Set gravity (default: -9.81)
- Configure default material
- Set solver iterations

## Input Manager

### Axes
- Horizontal: A/D or Left/Right arrows
- Vertical: W/S or Up/Down arrows
- Mouse X: Mouse X axis
- Mouse Y: Mouse Y axis
- Fire1: Left mouse button or Left Ctrl
- Fire2: Right mouse button or Left Alt
- Fire3: Middle mouse button or Left Shift
- Jump: Space
- Submit: Enter or Space
- Cancel: Escape

### Mobile Input
- Touch input for mobile platforms
- Accelerometer for tilt controls
- Gyroscope for advanced controls

- Player
